# IHK

- Bin bei IHK München

## Nutzen

- Interessenvertretung
- Prüfungen
- Schulungen

## Kosten

- Grundbetrag: 40€/a
- Umlage: 0,129% von allem Gewinn über 15.349€
- Außnahmen:
    - Gewinn < 5.200: => 0.0€
    - Gründer, 1es oder 2es Jahr: => 0.0€
    - Gründer, 3es oder 4es Jahr: => 0.0€ von Umlage, aber Grundbetrag schon.
